# 📊 Sales Performance Dashboard (EDA & Visualization)

## 📌 Project Overview

The **Sales Performance Dashboard** project focuses on analyzing sales data using **Exploratory Data Analysis (EDA)** techniques and creating meaningful visualizations to uncover business insights, sales trends, and performance patterns.

This project helps in understanding customer behavior, product performance, and regional sales distribution through data-driven analysis.

---

## 🎯 Project Objectives

* Analyze and explore sales data.
* Identify monthly sales trends.
* Compare product-wise sales performance.
* Analyze region-wise sales distribution.
* Visualize data using different charts and graphs.
* Generate meaningful business insights for decision-making.

---

## 🛠️ Tools & Technologies Used

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**

---

## 📂 Dataset Information

The dataset contains the following columns:

| Column Name  | Description                  |
| ------------ | ---------------------------- |
| Date         | Date of the sale             |
| Product      | Name of the product sold     |
| Region       | Sales region                 |
| Sales_Amount | Revenue generated from sales |
| Quantity     | Number of units sold         |

## 🔍 Tasks Performed

### 1. Data Loading

* Imported the dataset using Pandas.
* Checked dataset structure and dimensions.

### 2. Data Exploration (EDA)

### 3. Data Visualization

Created the following visualizations:

📈 **Monthly Sales Trend Analysis**

📉 **Histogram for Sales Distribution**

🔵 **Scatter Plot for Relationship Analysis**

📦 **Box Plot for Outlier Detection**
