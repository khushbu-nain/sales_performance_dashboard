# sales_performance_dashboard
#Sales Performance Dashboard (EDA & Visualization)

## Project Objective
Analyze a sales dataset using Exploratory Data Analysis (EDA) and create visualizations to discover sales trends and insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Columns
- Date
- Product
- Region
- Sales_Amount
- Quantity

## Tasks Performed
1. Loaded the dataset.
2. Performed initial EDA.
3. Created monthly sales trend.
4. Analyzed product-wise and region-wise sales.
5. Created scatter plot, histogram, and box plot.
6. Summarized key insights.

## How to Run
1. Install required libraries.
2. Open `Sales_Performance_Dashboard.ipynb`.
3. Run all cells in order.
